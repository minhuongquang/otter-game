# Environment

Target:

- Windows 11
- Godot 4.x stable
- GDScript 2.0
- PowerShell
- Git and GitHub

Use PowerShell commands and Windows paths for local instructions.

Inside Godot, prefer `res://` paths.

Do not use Godot 3 APIs or C# unless explicitly requested.

When a command depends on a local Godot executable path, inspect the repository or ask only if it cannot be inferred.
